文件整理工具使用说明

1. 首次运行说明：
   - 双击运行 file_organizer.bat
   - 首次运行时会自动下载便携版Python环境（约500MB）
   - 下载和解压过程可能需要几分钟，请耐心等待

2. 后续使用：
   - 直接双击 file_organizer.bat 即可启动程序
   - 程序会自动使用已下载的Python环境

3. 注意事项：
   - 请确保电脑连接网络（仅首次运行需要）
   - 请不要删除 portable_python 文件夹
   - 如果程序无法启动，可以尝试删除 portable_python 文件夹后重新运行

4. 文件说明：
   - file_organizer.bat: 启动程序
   - file.py: 主程序文件
   - portable_python/: Python运行环境（首次运行后自动创建）

如有问题请联系开发者。
